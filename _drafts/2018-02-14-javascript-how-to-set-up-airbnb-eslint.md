<h3><a href="https://en.wikipedia.org/wiki/Lint_%28software%29" target="_blank">Lint</a></h3>Definition taken from Wikipedia: <br />A lint refers to tools that analyze source code to flag programming errors, bugs, stylistic errors, and suspicious constructs. <br /><br />In other word, it is a tool that scans your project files and point out potential issues such as formatting issue, reference to a non-existent function, unused variables etc.<br /><h3>JavaScript Linting - ESLint</h3><a href="https://eslint.org/" target="_blank">ESLint</a> is a lint tool for JavaScript.
