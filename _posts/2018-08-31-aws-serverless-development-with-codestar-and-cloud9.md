Learn how to use AWS Cloud IDE CodeStar.   

I came to know about <a href="https://aws.amazon.com/cloud9/" target="_blank">Cloud9</a>:&nbsp;A cloud IDE for writing, running, and debugging code when I started using <a href="https://aws.amazon.com/codestar/" target="_blank">CodeStar</a>: a tool that streamlines the usage of multiple aws services such as<br />i)&nbsp;<a href="https://aws.amazon.com/codecommit/" target="_blank">AWS CodeCommit</a>&nbsp;- a fully-managed source control service that makes it easy for companies to host secure and highly scalable private Git repositories.<br />ii)&nbsp;<a href="https://aws.amazon.com/codebuild/" target="_blank">AWS CodeBuild</a>&nbsp;-&nbsp;a fully managed build service that compiles source code, runs tests, and produces software packages that are ready to deploy<br />iii)&nbsp;<a href="https://aws.amazon.com/cloudformation/" target="_blank">AWS CloudFormation</a>&nbsp;-&nbsp;Model and provision all your cloud infrastructure resources<br />iv) <a href="https://aws.amazon.com/cloudwatch/" target="_blank">Amazon CloudWatch</a> -&nbsp;a monitoring and management service built for developers, system operators, site reliability engineers (SRE), and IT managers.<br />v) <a href="https://aws.amazon.com/cloud9/" target="_blank">AWS Cloud9</a> -&nbsp;a cloud-based integrated development environment (IDE) that lets you write, run, and debug your code with just a browser.<br />The descriptions above were taken from respective AWS webpages.<br /><br />You can check out some screenshots below on how to create a project using CodeStar.
