From the examples below, you can notice that performance.now() returns a more precise value which is of  DOMHighResTimeStamp type. It is accurate to 5 microseconds.

performance.now()
0.5999999702908099

console time

0.6279296875
You can replace the fetch block with the block of code that you want to measure such as processing JSON data. 

Console.time()

console.time('fetching ip info');
fetch('http://ip-api.com/json/google.com')
  .then(data => data.json())
  .then(data =>  {
    console.log(data);
  });
console.timeEnd('fetching ip info');



Performance.now()


var t0 = performance.now();
fetch('http://ip-api.com/json/google.com')
  .then(data => data.json())
  .then(data =>  {
    console.log(data);
  });
var t1 = performance.now();
console.log("Call to fetch ip info took " + (t1 - t0) + "milliseconds.");
